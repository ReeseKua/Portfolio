Lesson 1: CSS Basics

CSS controls the style and layout of web pages. I learned to set background colors, text fonts, and align content properly.

What I learned: I can now change the appearance of a webpage and make it visually appealing.

Lesson 2: Style Sheets

Learned inline, internal, and external CSS. Using different types helps manage website styling efficiently.

What I learned: I understand the difference between inline, internal, and external CSS and when to use each.

Lesson 3: Display Property & Span/Div

Used block and inline elements, learned div for sections and span for inline text. Helps organize layout.

What I learned: I can control the placement of elements using block and inline, and use div/span effectively.

Lesson 4: CSS Rule & Class Selector

Learned how to structure rules and use classes for reusable styling. Grouping selectors improves code efficiency.

What I learned: I can create reusable styles for multiple elements using classes and make my CSS cleaner.

Lesson 5: CSS Selectors

Studied element, class, ID, universal, group, and attribute selectors. Helps target elements precisely for styling.

What I learned: I can select specific elements efficiently to apply consistent styling across the page.

Lesson 6: Dimensions & Pseudo-Classes

Controlled size, visibility, and added hover effects. Pseudo-classes make web pages more interactive and dynamic.

What I learned: I can adjust element size, visibility, and make links and buttons interactive for better UX.

My School Events

Buwan ng Wika

During the Buwan ng Wika celebration, I learned the importance of our language in expressing our culture, traditions, and identity as Filipinos. I can apply this in real life by actively using Filipino in conversations, school projects, and by showing respect for our cultural heritage. I participated actively in the event by attending competitions, watching presentations, and engaging in school programs. If I were to teach this topic to a classmate, I would explain that language is not just a tool for communication but also a way to preserve and understand our culture. This event is important because it inspires students to appreciate and take pride in their own language and identity as Filipinos.
